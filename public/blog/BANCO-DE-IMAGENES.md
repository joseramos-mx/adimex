# Banco de imágenes — Blog ADIMEX

Guía para las portadas (`cover`) de los 7 artículos del blog.

Ahora mismo cada post ya tiene una **portada de marca lista y funcionando** (ilustración
vectorial oscura con el estilo de ADIMEX, azul `#017bfd`). Están en `public/blog/` con el
nombre exacto que cada post espera, así que **no hay que tocar código**: el blog ya se ve.

Este documento es para cuando quieras **cambiar esas ilustraciones por fotos reales**.
Abajo tienes, para cada artículo, fotos reales seleccionadas — todas de **Pexels**, con
**licencia gratuita para uso comercial y sin atribución** (ver licencia al final).

---

## Cómo reemplazar una portada por una foto real

1. Abre el enlace de la foto que más te guste (columna "Opciones").
2. En Pexels pulsa el botón verde **"Free Download"** → elige tamaño **Large (1920px)**.
3. Renombra el archivo descargado con el **nombre exacto** de la columna "Archivo destino".
4. Cópialo dentro de `adimex/public/blog/` **sobrescribiendo** la ilustración actual.
5. Listo — el post lo toma automáticamente (misma ruta, mismo nombre).

> Recomendado: recorta la foto a **16:10** (1600×1000) antes de subirla para que encaje
> igual que las ilustraciones actuales y no se deforme en las tarjetas del blog.
